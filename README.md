# IBGP · Datas dos Concursos

App para visualizar prazos de inscrição e isenção dos concursos direto do Microsoft Planner.

## Deploy no Streamlit Cloud

1. Suba este repositório no GitHub
2. Acesse [share.streamlit.io](https://share.streamlit.io)
3. Conecte o repositório e selecione `app.py`
4. Após publicar, copie a URL do app (ex: `https://ibgp-planner.streamlit.app`)
5. No Azure Portal, adicione essa URL como Redirect URI no app "IBGP Planner Bot"

## Uso

Acesse o app e clique em "Entrar com Microsoft" — faça login com sua conta do IBGP.
